WINDKESSEL MODEL — OTTO FRANK 1899

نموذج Windkessel الشرياني

محاكاة رياضية للضغط الشرياني بمعادلة تفاضلية من الدرجة الأولى

نموذج Windkessel ذو العنصرين
C · dP(t)/dt = Q(t) − P(t)/R
Q(t) = النبضة القلبية (L/min)   P(t) = الضغط الشرياني (mmHg)   R = مقاومة الأوعية المحيطية   C = الامتثالية الشريانية الكلية
R — مقاومة
1.5
mmHg·s/mL
C — امتثالية
2.0
mL/mmHg
ثابت الزمن τ = R × C
τ = 3.00 ثانية
زيادة R (تصلب الشرايين) = ارتفاع ضغط الدم الانبساطي
انخفاض C (عمر أكبر) = ضغط نبضي أعلى = نمط LVH في ECG
المرجع: Garon A, Caron M. "Modèle mathématique de la circulation sanguine." Accromath. 2007;2:2-7.
الحل العددي — Euler Method
windkessel_simulate.m
% حل معادلة Windkessel بطريقة Euler العددية dt = 0.01; T = 10; N = T/dt; P = zeros(1,N); P(1) = 80; % mmHg إبتدائي for i = 2:N Q_t = Q_cardiac(t(i)); % نبضة القلب dP = (Q_t/C - P(i-1)/(R*C))*dt; P(i) = P(i-1) + dP; end
